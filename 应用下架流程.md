应用下架流程

```shell
1. 删除域名的Nginx配置和SSL证书
	到proxy-env项目，将该项目的配置文件复制到 .env
	执行 sh remote-delete.sh

1. 删除域名解析

2. 删除域名

3. 删除所有App

4. 删除项目代码

# 将xz-xxx 替换为项目名字
project_name=xx-xxx
cd /hk/${project_name}/env
docker-compose down
cd ../../
rm -rf /hk/${project_name}
rm -rf /hk/.${project_name}.*
rm -rf /root/.laradock/${project_name}
```
